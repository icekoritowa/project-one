# Мой первый проект

## Проект
Этот репозиторий содержит форму обратной связи с модальным окном и валидацией.

## Функциональность
- Модальное окно с формой обратной связи
- Валидация полей (имя, email, телефон, тема, сообщение)
- Маска для телефона
- Доступность с клавиатуры
